# koa-react-ssr-starter

> to understand && to explain how react ssr works.

服务端渲染的React手脚架

这个REPO的设计是为了让你使用一整套最新最酷的前端技术，所有都是可配置，富特性，基于webpack已经提供代码热加载，使用postcss预处理css，mobx-state-tree 等等更多

所有相关库已准备好，随时等待调用

觉得不错的话, 欢迎star

## installation

```
$ npm i

or

$ yarn
```

## run
```
npm run dev
```

## support
- [x] ssr
- [x] hmr
- [x] write api
- [x] connect mongodb

## stack:
- koa2
- react16
- react-router4
- webpack3
- mobx
- postcss
